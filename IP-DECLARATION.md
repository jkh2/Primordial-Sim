1. Primordial-Sim (Genesis Engine)
A real-time WebGL particle ecosystem simulator featuring:
Genesis Engine — A continuous-space organism simulation engine using Structure of Arrays (SoA) data architecture, spatial hash grid collision detection, and five behavioral force drives (hunt, flee, flock, food attraction, separation) operating on up to 60,000 entities at 60fps in a web browser
Genetic Evolution System — Four-gene heritable trait system (Speed, Aggression, Efficiency, Perception) with configurable mutation rate and strength, producing emergent natural selection and species divergence without scripted fitness functions
Food Chain Cycle System — Rock-Paper-Scissors circular predation mode creating Lotka-Volterra population oscillations
Food Oasis Geography — Configurable fertile zones that bias resource distribution and create migration patterns and territorial competition
Death Particle System — Visual kill/death feedback using species-colored particle bursts
Procedural Audio Ecosystem — Web Audio API ambient soundscape that responds dynamically to population levels, reproduction events, and predation kills
Scenario Preset Architecture — Comprehensive parameter preset system setting 20+ simulation variables simultaneously to create distinct ecosystem experiences
2. AI Lab Partner Integration
A novel system for integrating AI as an active research participant in a running simulation:
State Snapshot Engine — Real-time capture of full simulation state including per-species population counts, gene averages, extinction events, predator statistics, and parameter settings, structured as context for AI analysis
Experiment Engine — Automated experiment execution system that parses AI-generated JSON protocols, programmatically applies simulation parameter changes, runs timed experiments with before/after data capture, and triggers AI analysis on completion
Multi-Provider AI Adapter — Unified API layer supporting OpenAI, Anthropic, xAI (Grok), and any OpenAI-compatible local endpoint (LM Studio, Ollama, vLLM, etc.) with in-browser API key storage and automatic format translation between provider-specific message formats
AI-Driven Scientific Methodology — System enabling AI to perform the full scientific method cycle against a live simulation: observation, hypothesis formation, experiment design with specific parameters, automated execution, data collection, comparative analysis, report generation, and follow-up experiment suggestion
